# X AI回复生成器 - 安装指南

## 快速安装（5分钟）

### 第一步：准备API密钥

1. 访问 [OpenAI平台](https://platform.openai.com/api-keys)
2. 登录或注册账号
3. 点击"Create new secret key"创建新的API密钥
4. 复制并保存API密钥（格式类似：`sk-...`）

### 第二步：安装Chrome插件

#### 方法一：从文件夹安装（推荐）

1. 解压 `x-reply-generator.zip` 到任意位置
2. 打开Chrome浏览器
3. 在地址栏输入：`chrome://extensions/` 并回车
4. 在页面右上角启用"**开发者模式**"（Developer mode）
5. 点击左上角的"**加载已解压的扩展程序**"（Load unpacked）
6. 选择解压后的 `x-reply-generator` 文件夹
7. 点击"选择文件夹"
8. 插件安装完成！你会在Chrome工具栏看到插件图标

#### 方法二：直接从ZIP文件安装

1. 打开Chrome浏览器
2. 在地址栏输入：`chrome://extensions/` 并回车
3. 在页面右上角启用"**开发者模式**"
4. 将 `x-reply-generator.zip` 拖拽到Chrome扩展页面
5. 如果需要，解压并按方法一操作

### 第三步：配置插件

1. 点击Chrome工具栏中的插件图标 ✨
2. 在弹出窗口中点击"**⚙️ 打开设置**"
3. 在"OpenAI API密钥"输入框中粘贴你的API密钥
4. （可选）选择AI模型：
   - **GPT-4 Mini**（推荐）：成本低，质量高
   - **GPT-4 Nano**：超低成本
   - **Gemini 2.5 Flash**：Google模型
5. （可选）设置自定义提示词，或点击快速模板按钮
6. 点击"**💾 保存设置**"
7. 看到"✅ 设置已保存"提示即表示配置成功

### 第四步：开始使用

1. 访问 [X (Twitter)](https://x.com)
2. 浏览任意推文
3. 在推文下方找到新增的"**✨ AI回复**"按钮
4. 点击按钮，等待几秒
5. 生成的回复会自动加载到回复输入框
6. 编辑回复（如需要）
7. 点击发送按钮发布回复

## 常见问题排查

### 问题1：找不到"加载已解压的扩展程序"按钮
**解决方案**：确保已启用"开发者模式"（页面右上角的开关）

### 问题2：插件安装后没有图标
**解决方案**：
1. 检查 `chrome://extensions/` 页面，确认插件已启用
2. 点击Chrome工具栏右侧的拼图图标
3. 找到"X AI回复生成器"并点击固定图标

### 问题3：点击"AI回复"按钮没有反应
**解决方案**：
1. 检查是否已设置API密钥
2. 打开浏览器控制台（F12）查看错误信息
3. 确认网络连接正常
4. 刷新X页面后重试

### 问题4：提示"API错误"
**解决方案**：
1. 检查API密钥是否正确
2. 确认API密钥有足够的额度
3. 访问 [OpenAI使用情况](https://platform.openai.com/usage) 查看额度

### 问题5：生成的回复没有加载到输入框
**解决方案**：
1. 回复会被复制到剪贴板
2. 手动粘贴（Ctrl+V 或 Cmd+V）到回复框
3. 如果问题持续，请刷新页面

## 卸载插件

1. 访问 `chrome://extensions/`
2. 找到"X AI回复生成器"
3. 点击"移除"按钮
4. 确认删除

## 隐私说明

- API密钥存储在Chrome本地，不会上传到任何服务器
- 推文内容仅发送到OpenAI API用于生成回复
- 插件不会收集或存储任何个人数据

## 技术支持

如遇到其他问题，请：
1. 查看浏览器控制台（F12）的错误信息
2. 检查 `README.md` 中的详细文档
3. 确保Chrome浏览器版本为最新

## 更新插件

当有新版本时：
1. 下载新版本的ZIP文件
2. 解压到原位置（覆盖旧文件）
3. 访问 `chrome://extensions/`
4. 点击插件卡片上的"刷新"图标
5. 更新完成

## 系统要求

- Chrome浏览器 88+ 或 Edge 88+
- 有效的OpenAI API密钥
- 稳定的网络连接
